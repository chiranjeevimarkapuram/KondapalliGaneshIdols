# KondapalliGaneshIdols


Kondapalli Ganesh Idols Website

A responsive multi-page website built to showcase eco-friendly Kondapalli clay Ganesh idols crafted using traditional Andhra Pradesh artistry.

This project highlights cultural heritage, promotes sustainable festival celebrations, and provides a simple way for customers to order idols via WhatsApp.

Project Overview

This website is designed to:

Promote handmade Kondapalli clay idols

Highlight eco-friendly Ganesh idol collections

Showcase the 2025 special collection

Provide location and contact information

Enable direct ordering through WhatsApp integration

The project is built using basic front-end technologies and Bootstrap for responsive design.

Features

Multi-page website (Home, About, 2025 Collection, Contact)

Responsive layout using Bootstrap 5

WhatsApp order integration

Image gallery for idol showcase

Clean navigation bar across all pages

Cultural and eco-friendly theme

Technologies Used

HTML5
CSS3
Bootstrap 5
GEN AI

WhatsApp API Link Integration
